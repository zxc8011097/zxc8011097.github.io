# ⚽ 足球比分AI预测系统 · 专业增强版

> 基于 football-data.org API 的实时赛事数据与智能预测工具，支持多联赛、AI比分预测、PWA 桌面安装。

---

## 🎯 项目简介

本项目是一个**免费、开源的足球比分AI预测系统**，通过对接真实赛事 API，提供：
- 实时全球足球赛事比分（英超、西甲、德甲、意甲、法甲、欧冠等）
- AI 智能比分与赛果预测
- 专业级深色界面，适配手机/平板/电脑
- 支持 PWA 安装，可添加到手机桌面，像原生 App 一样使用

🌐 **访问地址**：https://www.football-predict.top

---

## ✨ 核心功能

| 功能 | 说明 |
| :--- | :--- |
| 🔴 实时比分 | 自动拉取今日全球足球赛事，支持多联赛 |
| 🧠 AI 超能预测 | 基于泊松模型与球队实力，智能预测比分与赛果 |
| 📊 专业数据 | 射门、射正、控球率、传球率等实时统计 |
| 🎨 多主题 | 深色专业版 / 清爽版 / 极简版一键切换 |
| 📱 PWA 支持 | 可安装到手机桌面，离线缓存，体验更流畅 |
| 📈 专业对比 | 内置凯利指数、赔率计算、机构胜率对比 |
